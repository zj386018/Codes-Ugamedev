# 神秘森林文字冒险游戏 - 使用说明

## 🎮 游戏简介

这是一款基于Python tkinter开发的桌面版文字冒险游戏。玩家将在神秘的森林中探索，做出选择，收集物品，体验不同的故事结局。

## 🚀 启动游戏

### 🌟 推荐方式（简化版 - 最稳定）
```bash
python main_simple.py
```

### 备选方式（修复版）
```bash
python main_fixed.py
```

### 原始版本
```bash
python main.py
```

### 测试模式
```bash
python test_game.py
```

### 版本说明
- **简化版**：完全避免ttk主题问题，最稳定
- **修复版**：修复了大部分兼容性问题
- **原始版**：功能最完整但可能有兼容性问题

## 🎯 游戏特色

- **完整剧情**：6个精心设计的场景，分支剧情和多重结局
- **物品系统**：6种可收集和使用的物品
- **存档功能**：随时保存和加载游戏进度
- **音效反馈**：点击、获得物品等音效
- **现代UI**：深色主题，森林风格设计

## 🕹️ 操作说明

### 基本操作
1. **阅读场景描述**：在左侧文本框中查看当前场景
2. **做出选择**：点击下方的选择按钮推进剧情
3. **管理背包**：在右侧查看和使用物品
4. **保存进度**：使用底部按钮保存/加载游戏

### 物品使用
- 点击背包中的物品查看描述
- 点击"使用物品"按钮使用选中的物品
- 某些物品可以恢复生命值
- 关键物品可能影响剧情发展

### 控制按钮
- **🔄 重新开始**：重置游戏到初始状态
- **💾 保存游戏**：保存当前进度到文件
- **📂 加载游戏**：从文件读取保存的进度
- **❌ 退出游戏**：关闭游戏窗口

## 🗺️ 场景攻略

### 1. 森林入口
- 游戏开始地点
- 选择不同的探索方向

### 2. 小木屋
- 可能发现有用的物品
- 寻找线索和提示

### 3. 神秘洞穴
- 充满未知的危险
- 需要合适的装备

### 4. 古老遗迹
- 解谜和挑战
- 隐藏着重要秘密

### 5. 湖泊小岛
- 最终的考验
- 决定游戏结局

- **好结局**：找到宝藏，成为传奇
- **坏结局**：冒险失败，需要重新尝试

## 💡 游戏提示

1. **谨慎选择**：每个选择都可能影响后续剧情
2. **合理使用物品**：在关键时刻使用合适的物品
3. **及时存档**：在重要节点保存进度
4. **探索所有选项**：不同选择可能带来不同结果
5. **注意生命值**：生命值归零会导致游戏失败

## 🐛 故障排除

### 常见问题

1. **游戏无法启动**
   - 检查Python版本是否为3.6+
   - 确认tkinter库已安装
   - 尝试使用 `main_fixed.py` 启动

2. **界面显示异常**
   - 确保屏幕分辨率支持1000x700窗口
   - 检查系统DPI设置

3. **音效无法播放**
   - 检查系统音量设置
   - 确认Windows支持beep音效

4. **存档丢失**
   - 检查是否有写入权限
   - 确认存档文件未被删除

### 错误信息
如果遇到错误，请记录错误信息并检查：
- 所有游戏文件是否完整
- Python环境是否正确配置
- 是否有足够的系统权限
